RIDESHARE

Our Ride Sharing Service Features

Web-Based Ride Sharing

This application provides a complete ride-sharing (carpooling) service implemented as a web app.

Flexible Journey Options

Users have two primary functions: they can post a new route they plan to travel, or they can join a trip that is already planned or currently in progress.

Affordable, Non-Profit Focus

The core goal is to facilitate cheap, mutually beneficial travel. This commitment ensures the service is not used to maximize profit, maintaining its purpose as a true carpooling solution.

Unique Automated Fee Calculation

To ensure fair pricing, the application automatically calculates the fee. Drivers cannot set their own prices, eliminating price speculation.

Real-Time Live Location Sharing

A key feature is live location sharing (requires GPS). This allows both driver and passenger to select the most convenient partner, improving meeting efficiency and accuracy.

Dynamic Cost Recalculation

The fare is determined by distance in kilometers, the car's consumption, the car's price and model year.
